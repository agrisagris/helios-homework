## Prerequisites

For Next.js, Node.js version "^18.18.0 || ^19.8.0 || >= 20.0.0" is required.

## Running application

Install node modules:

```bash
npm install
# or
yarn install
```

Running application:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
